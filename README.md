# QBot
An 8DOF quad robot
## {Work in Progress}
The design and instructions in this document probably need further testing and optimization.
## {Introduction}
These are instructions for building a four legged robot (8 degrees of freedom). Designed for/during the Marietta College course "DIY Manufacture" during spring semester 2022.  Loosely based on "Kame"
 
## {Tools and materials}

	Arduino Nano Every
	headers	
	cable (USB Micro-B)
	printed circuit prototyping board
	micro servo motors 8 (9g, SG90)
	battery
	DC-DC converter/power conditioner
